== Description ==
A simple plugin to change all WooCommerce product prices to have a .99 suffix. 
This is useful for adjusting prices in bulk without having to edit each product individually.

== Installation ==
1. Upload the `woocommerce-price-changer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Price Change' menu item in the admin dashboard.
4. Click the 'Change Prices' button to update all product prices.

== Frequently Asked Questions ==

= Is this plugin safe to use? =
Yes, the plugin is designed to update prices safely. However, always backup your database before making bulk changes.

= Will this change all my product prices? =
Yes, this plugin will change all WooCommerce product prices to have a `.99` suffix based on their current prices.

= Can I use this on any WordPress site? =
This plugin requires WooCommerce to be installed and activated on your WordPress site.
